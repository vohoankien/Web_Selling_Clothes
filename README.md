# Website và Phần Mềm Quản Lý Bán Quần Áo

## Giới Thiệu

Dự án **Xây dựng phần mềm - website bán quần áo** là một nền tảng trực tuyến kết hợp phần mềm quản lý giúp các cửa hàng quần áo quản lý thông tin sản phẩm, khách hàng và đơn hàng hiệu quả. Đồng thời, nó cũng cung cấp trải nghiệm mua sắm trực tuyến hiện đại, an toàn và tiện lợi cho khách hàng.

## Tính Năng Chính

### Website dành cho khách hàng:
- **Quản lý tài khoản**: Đăng ký, đăng nhập và cập nhật thông tin cá nhân.
- **Tìm kiếm và sắp xếp sản phẩm**: Tìm sản phẩm theo từ khóa, danh mục hoặc lọc theo giá và đánh giá.
- **Đặt hàng trực tuyến**: Thêm sản phẩm vào giỏ hàng, thanh toán an toàn qua nhiều phương thức.
- **Theo dõi trạng thái đơn hàng**: Xem thông tin sản phẩm đã đặt, trạng thái đơn hàng và lịch sử mua sắm.
- **Đánh giá sản phẩm**: Gửi phản hồi và đánh giá chất lượng sản phẩm đã mua.

### Phần mềm dành cho quản trị viên và nhân viên:
- **Quản lý sản phẩm**: Thêm, sửa, xóa sản phẩm và danh mục.
- **Quản lý khách hàng**: Lưu trữ, chỉnh sửa thông tin và phân quyền tài khoản khách hàng.
- **Quản lý đơn hàng**: Theo dõi và xử lý trạng thái đơn hàng.
- **Thống kê và báo cáo**: Xuất báo cáo doanh thu, xu hướng mua sắm và sản phẩm bán chạy.
- **Quản lý thanh toán**: Xử lý các giao dịch thanh toán trực tuyến và khi nhận hàng.

## Công Cụ Sử Dụng

- **Phân tích và thiết kế**: IBM Rational Rose, BPMN.io.
- **Ngôn ngữ lập trình**: C#.
- **Môi trường phát triển**: Visual Studio 2022, Visual Studio Code.
- **Cơ sở dữ liệu**: SQL Server.

## Hướng Dẫn Cài Đặt

1. **Yêu cầu hệ thống**:
   - .NET Framework 4.7 trở lên.
   - SQL Server 2019 hoặc cao hơn.
   - Visual Studio 2022.

2. **Cài đặt**:
   - Clone repo về máy:
     ```bash
     git clone https://github.com/nhom10-quanlybanhang/nhom10-banquanao.git
     ```
   - Import cơ sở dữ liệu từ tệp backup có sẵn trong thư mục `Database`.
   - Mở dự án trong Visual Studio và chạy chương trình.

3. **Cấu hình**:
   - Cập nhật chuỗi kết nối cơ sở dữ liệu trong tệp cấu hình ứng dụng.
